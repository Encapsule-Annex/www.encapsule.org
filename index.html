2013.02.04 update

I've been actively working on a web-based editing application for Soft Circuit Description Language (SCDL) models. SCDL (pronounced "scuddle") is a JSON object model for "soft circuits." Soft circuits are system design artifacts conceptually analogous to digital circuit schematics.

The application is called Schema and should be online and functional sometime late in February, 2013.

In addition to being a cool HTML5 demo, I am excited about the SCDL editor because SCDL models are machine-readable system specifications that can be used like blueprints.

  • Quickly 'gist' SCDL models and share via GitHub.
  • Interpret SCDL at runtime. (e.g. in Javascript in a generic host implemented in Javascript)
  • Transform SCDL to something else and compile or interpret that. (e.g. a SCDL model can be transformed into a C++ meta program).
  • Leverage SCDL for system simulation, verification, and test automation.

Encapsule Project
Working to make programming accessible to everyone via
open source models, and visual programming tools for the web.

Code: GitHub/Encapsule • Blog: blog.encapsule.org
Twitter: @Encapsule • Schema: Schema SCDL Editor

Encapsule Project is an open source project focused on the development of infrastructure libraries, development tools, and utilities for building complex software systems from visual models that are understandable and accessible to everyone via the modern browser.

The core technologies being developed are a fusion ideas from digital circuit design, component-based engineering, dataflow programming, graph theory, domain-specific languages, and data visualization.
